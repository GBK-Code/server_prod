### Server for the Idea notebook project
## Testing stage v.0

Idea notebook is a project that works on Qt/QML client and allows users report to developers about ideas and/or bugs.

## * How it works?

Users fill a form via link where they input title and details of idea/bug. This report goes to this server that includes it into database. 
A developer with the client can see these reports in the GUI window. Right now such operations for the client as:

* Refresh
* Delete report
* Look into report details

avaiable
